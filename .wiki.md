# Project Summary
AccessScan AI is an AI-powered website accessibility scanner that assists businesses in identifying and resolving compliance issues with WCAG standards. The platform features a user-friendly interface and advanced scanning capabilities, integrating payment processing through Stripe for subscription management. By providing detailed reports and actionable insights, AccessScan AI helps mitigate legal risks associated with website accessibility, making it essential for businesses aiming to enhance their digital presence and inclusivity.

# Project Module Description
The project consists of several functional modules:
- **Landing Page**: Showcases product features and user subscription status.
- **Scan Feature**: Users can input a URL to generate comprehensive accessibility reports.
- **Pricing Tiers**: Offers various subscription plans, including a free tier for initial scans.
- **Results Page**: Displays scan results, highlighting accessibility issues and compliance scores.
- **Success Page**: Confirms successful payments and provides next steps for Pro users.
- **Responsive Design**: Ensures usability across devices.
- **Backend Integration**: Manages user subscriptions, scan history, and payment processing through Stripe.
- **Edge Functions**: Implements server-side logic for subscription handling and website scanning, including webhooks for subscription updates.

# Usage
To set up the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/hustleforge/accessscan-ai.git
   cd accessscan-ai
   ```
2. Install dependencies: 
   ```bash
   pnpm install
   ```
3. Set up environment variables: Create a `.env` file with the following content:
   ```env
   VITE_SUPABASE_URL=https://gdtpziynhssqytcnzvbi.supabase.co
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```
4. Run the development server:
   ```bash
   pnpm run dev
   ```
